# Numero Secreto

Una aplicación web simple para adivinar un número secreto del 1 al 10.  
Permite ingresar números aleatorios e indica si el número correcto es mayor o menor.

---

## 🚀 Características

- Agregar nombres de amigos a la lista.
- Validación: no se permite agregar campos vacíos.
- Visualización dinámica de los nombres ingresados.
- Sorteo aleatorio de un amigo secreto usando `Math.random()` y `Math.floor()`.
- Interfaz sencilla y responsive.

---

- **CSS3**  
- **JavaScript (Vanilla JS)**

## 📖 Cómo usar

1. Abre el proyecto en el link https://mcamijar.github.io/juegoSecreto/.

3. Ingresa el número que quieras adivinar y presiona **Intentar**.

4. En pantalla aparecerá si el número correcto es mayor o menor a tu intento.

5. ¡Sigue intentando hasta lograrlo!
